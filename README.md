# Шутник

Веб-приложение, которое шутит.

## Использование

Приложение отвечает на запросы:

    GET /joke

И выдаёт смешные шутки.

## Ограничения

Придумывать шутки нелегко, поэтому приложение может выдавать только 5 шуток в минуту.

## Используемые технологии

* Ruby on Rails
* RSpec
* PostgreSQL
* Redis (Необязательно)
